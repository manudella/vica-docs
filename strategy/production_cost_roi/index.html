    Qwen2.5-VL-72B OCR ~80GB +40–50% accuracy

    Model-agnostic architecture

    Swap models with zero code changes — only configuration updates. The system supports any OpenAI-compatible model.

    Operational KPIs¶

    KPI Target Frequency
    Cost per CV screen < $8 Monthly
    Time to shortlist < 2 business days Per job
    Recruiter throughput 50–100 CVs/day Weekly
    System availability > 99.5% Monthly
    Avg CV processing time < 3 min end-to-end Weekly

    Quality KPIs¶

    KPI Target Measurement
    Precision@K > 0.70 Compare top-K to actual hires
    Recall@K > 0.65 Coverage of positive candidates
    F1 Score > 0.65 Harmonic mean
    Skill coverage > 85% % of job skills with matches

    Business KPIs¶

    KPI Target Frequency
    Cost per hire reduction > 50% Quarterly
    Time-to-fill reduction > 20% Quarterly
    Recruiter satisfaction > 4.0/5.0 Quarterly
    Diversity of shortlists Maintain or improve Semi-annually

    Risk Matrix¶

    Risk Probability Impact Mitigation
    GPU hardware failure Low High Warranty, CPU fallback mode
    Model quality for niche roles Low–Med Medium Upgrade to 70B; custom fine-tuning
    Key person dependency Medium High Runbooks, cross-training
    Recruiter adoption resistance Medium Medium Phased rollout, champion users
    Integration complexity overrun Medium Medium Agile sprints, 15% contingency
    Scaling beyond single server Low Low–Med Multi-GPU upgrade path

    Implementation Roadmap¶

    Phase 1 — Discovery & Procurement (Weeks 1–2)
    Requirements doc, hardware ordered, vendor contracts. $2K–$5K
    Phase 2 — Foundation & Deployment (Weeks 3–6)
    Server live, VICA deployed, GPU validated, security hardened. $20K–$35K
    Phase 3 — Integration (Weeks 5–14)
    ATS connector, SSO, ERP link, PostgreSQL migration. $30K–$65K
    Phase 4 — Customization (Weeks 10–16)
    Consulting skill taxonomy, RBAC, branded reports. $15K–$35K
    Phase 5 — Validation (Weeks 14–20)
    Historical validation (F1 > 0.65), load testing passed. $8K–$15K
    Phase 6 — Pilot (Weeks 18–24)
    5–10 recruiters live, 2–3 real jobs processed, feedback collected. $5K–$10K
    Phase 7 — Full Rollout (Weeks 22–28)
    All recruiters trained, monitoring live, KPIs tracking. $10K–$15K

    Success Criteria for Go-Live¶

    • System processes 95%+ of uploaded CVs without errors
    • F1 score exceeds 0.65 in validation against historical hires
    • System handles peak load (50+ concurrent CVs) without degradation
    • Recruiter satisfaction averages above 4.0/5.0 in pilot
    • All critical integrations (ATS, SSO) operational
    • Monitoring, alerting, backup, and DR tested
    • Security audit passed with no critical findings

    Cost estimates based on Western European labor rates (€700–900/day), Q1 2026 hardware pricing, and industry benchmarks. 15% contingency included in Year 1.
