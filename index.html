    VICA — AI-Powered Recruitment Screening¶

    On-Premise. Explainable. Open Source.

    The first open-source AI recruitment screener with citation-level explainability — built for EU AI Act compliance.

    Quick Start For Recruiters For Developers

    What VICA Does¶

    graph LR
    A["📄 Upload CVs"] --> B["🔍 OCR & Parse"]
    B --> C["🧠 AI Skill Extraction"]
    C --> D["⚖️ SPEA2 Ranking"]
    D --> E["📊 Evidence Reports"]

    • Complete Data Sovereignty

      All data stays on your hardware. No cloud APIs, no third-party processors, no cross-border transfers. GDPR compliant by architecture.

    • Citation-Level Explainability

      Every skill assessment links to verbatim CV quotes. 4-level proficiency, justification scoring, and targeted interview questions.

    • SPEA2 Multi-Objective Ranking

      Pareto-optimal candidate selection across all job skills simultaneously — no information loss from single-score collapse.

    • EU AI Act Ready

      Open-source and auditable. Built-in validation framework (Precision, Recall, F1). Recruiter oversight loop satisfies Art. 13 & 14.

    • Zero License Fees

      Entire stack is open source: FastAPI, React, Ollama, vLLM. 60–85% lower TCO vs cloud alternatives over 5 years.

    • Smart GPU Orchestration

      Single GPU shared between OCR and LLM with automatic switching. No dual-GPU required — works on an RTX 4090.

    Technology Stack¶

    Layer Technology
    Backend Python 3.11+, FastAPI, Uvicorn, Pydantic
    Frontend React 18, TypeScript, Vite, @dnd-kit
    LLM Ollama (llama3.1:8b), OpenAI-compatible API
    OCR vLLM (Qwen2.5-VL-3B-Instruct), vision model
    Data In-memory store + JSON file persistence
    Bias Reduction Regex-based agenderization (gender-neutral language)
    Matching SPEA2 (NumPy), RapidFuzz, ESCO taxonomy
    Reports Jinja2 + WeasyPrint
    Deployment Docker, Docker Compose, Nginx

    Quick Start¶

    cd vica
python -m venv .venv && .venv/Scripts/activate
pip install -r requirements.txt
python scripts/run_test.py

    Seeds 100 fake CVs + 1 tech job. Uses phrase matching (fast, no AI).

    # Terminal 1: Backend
pip install -r requirements.txt
ollama pull llama3.1:8b && ollama serve
python run_backend.py

# Terminal 2: Frontend
cd frontend && npm install && npm run dev

    Open http://localhost:5173. Full AI pipeline with OCR + LLM.

    docker compose up -d --build
docker exec vica-ollama ollama pull llama3.1:8b

    Full stack at http://localhost:3000.
