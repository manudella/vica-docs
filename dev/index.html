    Data Flow: Full Pipeline¶

    sequenceDiagram
    participant U as Recruiter
    participant FE as Frontend
    participant API as Backend API
    participant GPU as GPU Orchestrator
    participant OCR as vLLM (OCR)
    participant LLM as Ollama (LLM)
    participant DB as Store

    U->>FE: Upload CVs
    FE->>API: POST /candidates/upload
    API->>DB: Store file + record

    U->>FE: Parse All
    FE->>API: POST /candidates/{id}/preprocess
    API->>GPU: ensure_ocr()
    GPU->>LLM: Unload model
    GPU->>OCR: Start container
    API->>OCR: Extract text (per page)
    API->>DB: Save anonymized text

    U->>FE: Extract Skills
    FE->>API: POST /candidates/{id}/extract-skills
    API->>GPU: ensure_llm()
    GPU->>OCR: Stop container
    GPU->>LLM: Load model
    API->>LLM: Deep skill extraction
    API->>DB: Save skills + evidence

    U->>FE: Run Match
    FE->>API: POST /match/run-and-analyze (SSE)
    API->>DB: Build skill matrix
    API->>API: SPEA2 ranking
    API->>GPU: ensure_llm()
    loop For each top-K candidate
        API->>LLM: Semantic analysis
        API-->>FE: SSE progress event
    end
    API->>DB: Save results
    API-->>FE: SSE done event
